# 💰 FinSight — Personal Finance Tracker

FinSight is a modern, full-stack **personal finance tracker** that helps users monitor income, manage expenses, track budgets, and visualize spending trends over time.

🚀 **Live Demo:** https://finsight-fxd4.onrender.com

📦 **Tech Focus:** Full-stack development, databases, APIs, UI/UX, data visualization


## 🎯 Why I Built This

I built FinSight to practice designing a **full-stack application**, focusing on scalable backend architecture, database modeling, clean UI design, and financial data visualization.


## ✨ Features

### 💸 Transaction Management
- Add, edit, delete, and categorize income & expenses  
- Track spending with descriptions and timestamps  

### 📊 Real-Time Financial Dashboard
- Net balance overview  
- Income vs expense breakdown  
- Monthly summaries  

### 📅 Monthly Filtering
- Filter financial history month-by-month using a calendar picker  

### 🧮 Budget Tracking
- Set spending limits by category  
- Visual alerts when approaching or exceeding budgets  

### 📈 Data Visualization
- **Category Pie Chart** --> Spending distribution  
- **12-Month Bar Chart** --> Income v.s. expense trends  

### 🔍 Search & Sorting
- Search transactions by category or description  
- Sort by date or amount


## 🧠 Technical Highlights

- Designed **RESTful API** using Express
- Modeled relational data with **PostgreSQL + Drizzle ORM**
- Implemented **schema validation** with Zod
- Built **responsive UI** with Tailwind CSS + shadcn/ui
- Optimized frontend performance using **Vite**
- Created reusable UI components and shared schemas

## ⚙️ Getting Started (Local Setup)

### 1️⃣ Install dependencies
```npm install```

### 2️⃣ Configure environment variables
Create a `.env` file:
```DATABASE_URL=your_database_connection_string```

### 3️⃣ Run development server
```npm run dev```

### 4️⃣ Build & run production
```
npm run build
npm start
```
